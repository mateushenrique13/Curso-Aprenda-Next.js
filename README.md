# Curso-Aprenda-Next.js
Aqui ficara salvo minha trajetoria de experiencia em aprender como funciona o next.js e construir um projeto do zero com esse curso no decorrer das aulas

Neste curso, vou construir um painel financeiro que tenha:

Uma página inicial pública.
Uma página de login.
Páginas do painel de controle protegidas por autenticação.
A possibilidade de usuários adicionarem, editarem e excluírem faturas.
O painel de controle também terá um banco de dados associado
